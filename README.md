## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** Supabase (PostgreSQL)
- **Auth:** JWT
- **Deployment:** Vercel + Render



 ## 🚀 Local Setup

### Backend
```bash
cd backend
npm install
# Create .env with: SUPABASE_URL, SUPABASE_ANON_KEY, JWT_SECRET, PORT
npm start
```

### Frontend
```bash
cd frontend
npm install
# Create .env with: VITE_BACKEND_URL, VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY
npm run dev
```

## 📋 Environment Variables

See `.env.example` files in backend and frontend folders for required variables.
